## 🏁 Getting Started <a id = "getting_started"></a>

### 💻 Prerequisites

✔ Node 9+
✔ Npm/Yarn

### Installing


```
yarn install

// dev
yarn dev

// build
yarn build
```
- Create .env.local file with credentials.
- Create firebase.ts file in the root dir with firebase credentials.
