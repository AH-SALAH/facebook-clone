## 🏁 Getting Started <a id = "getting_started"></a>

### 💻 Prerequisites

✔ Node 9+
✔ Npm/Yarn

### Installing


```
yarn install

// dev
yarn dev

// build
yarn build
yarn start
```
- Create .env.local file with credentials. (e.g. fb nextauth related)
- Add firebase cred in .env.local corresponding to the firebase.ts file env keys.
